Descripción del Proyecto
Pagina Web destinada a organizar tareas para grupos. Dedicada al scrum en un futuro.

A fin de navegar por las secciones de la página web, el usuario será requerido iniciar sesión o registrarse en caso de no contar con usuario o contraseña. En ambas opciones, una vez la página valide la autenticación del usuario, este será redirigido al inicio de la página web.

Para navegar por las secciones el usuario debe registrarse, y ahi recien le muestra el resto de la pagina. 

Los usuarios pueden realizar las siguientes accciones:

Crear tareas
Ver su perfil  con las tareas por realizar
Modificar, eliminar y completar tareas
Cerrar sesion
Iniciar sesion en caso de haber cerrado sesión
Ver tareas completadas
